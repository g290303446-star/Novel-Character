> 由 `extract-fiction-character` 生成或填空。原则：结论尽量带 Evidence；推断标置信度；未写明的设定进入 DoNotInfer/OpenQuestions，禁止下游扮演时“补完”。

| excerpts_scope | （本次使用的片段范围/页码/章节号，可选） |
| language | （中文/其他） |
| last_updated | （日期） |
