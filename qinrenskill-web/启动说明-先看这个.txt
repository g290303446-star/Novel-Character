========================================
  记忆系统（小说提取）— 云服务器极简步骤
========================================

【你要做的只有这几步】

1）用宝塔「文件」把整个压缩包上传到服务器（任意目录，例如 /www/wwwroot/）

2）在宝塔里对压缩包点「解压」

3）打开宝塔「终端」，输入下面两行（路径改成你解压后的实际文件夹）：

   cd /www/wwwroot/qinrenskill-web
   chmod +x start.sh && ./start.sh

   第一次会多等一会儿（自动装 Python 依赖）。

4）用手机或电脑浏览器打开：

   http://你的服务器公网IP:8000/

   把「你的服务器公网IP」换成买服务器时给你的 IP 数字。

5）如果打不开页面：

   · 宝塔面板 → 安全 → 添加端口规则：8000
   · 阿里云/腾讯云等 → 控制台 → 安全组 → 入站规则 → 放行 TCP 8000

6）关闭服务：在终端里按键盘 Ctrl + C

（以后每次想用，再进文件夹执行一次 ./start.sh 即可。）

========================================
常驻运行（可选）
========================================

不要关终端就掉线：在宝塔「软件商店」安装「Supervisor / 进程守护」，
启动命令填（路径改成你的）：

   /www/wwwroot/qinrenskill-web/.venv/bin/uvicorn server.main:app --host 0.0.0.0 --port 8000

运行目录：/www/wwwroot/qinrenskill-web
环境变量：COOKIE_SECURE=0

========================================
数字人视频 MVP（可选）
========================================

· 解压后把 .env.example 复制为 .env（与 start.sh 同级），编辑填写：
    DASHSCOPE_API_KEY=（北京地域百炼 API Key）
    WAN_S2V_PUBLIC_BASE_URL=http://你的服务器公网IP
  不要末尾斜杠；保存后重启 ./start.sh
· 压缩包内不含 .env（防泄露）；勿把 .env 上传到公开仓库。

========================================
说明（可选读）
========================================

· 不要用「本地双击打开 html」，必须用浏览器访问 http://IP:8000/
· 本包含：Python3.11 兼容修复、首页手机端标签布局、数字人 wan2.2-s2v、.env 加载等。
· 以后若有域名和 HTTPS，再把 COOKIE_SECURE 改成 1；只用 IP 时不要改。
