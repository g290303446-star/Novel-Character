# DeepSeek 一键生成扮演指令（桌面版）

## 这是什么（给普通用户的解释）

你只需要：

1) 输入 DeepSeek API Key（每次输入，不保存）  
2) 选择一份小说 `.txt`  
3) 填人物名称（可选别名）  

点击一次“开始生成”，程序会自动输出三份文件到 `artifacts/`：

- `角色片段-<人物>.md`：从整本书里抓到的所有相关片段（用于追溯证据）
- `人物档案-<人物>.md`：结构化的人物设定（口吻/人格/目标/关系/禁区等）
- `扮演指令-<人物>.md`：可直接放进“系统提示词”的扮演指令

**原则解释**：先“本地全书召回”（不漏），再“模型归纳+编译”（不乱编）。扮演指令会强调 HardBoundaries / DoNotInfer / 不确定性处理，降低模型补完设定的风险。

## 如何运行（开发/内测）

在仓库根目录运行：

```powershell
python -m app.main
```

若你用的是 `python app\main.py`，也可以，但推荐模块方式运行以保证相对导入正常。

## 常见问题

### 1) 生成很慢/花费较高

**原因**：本工具默认使用 `deepseek-reasoner`，更强但更慢更贵。

建议：先缩小别名数量（只留本名+最常用外号），减少片段数量；或后续我可以加一个“快速模式（deepseek-chat）”开关给普通用户。

### 2) 片段抽不出来

可能原因：

- 人物名/别名写错或不常用
- 小说 txt 编码异常（脚本已尝试 utf-8 与 gb18030，但仍可能有极端情况）

建议：先只填“人物本名”试一次；如果仍为 0，检查 txt 是否真的是纯文本。

### 3) API 报错（401/429/5xx）

- 401：Key 不对或无权限
- 429：限流，稍后重试
- 5xx：服务端波动，稍后重试

程序内置了对 429/5xx 的自动重试，但若持续失败，请检查网络或账户额度。

## 如何打包成 EXE（PyInstaller）

> 说明：打包步骤会随着你机器环境变化而变化，这里给“通用命令”。

1) 安装 PyInstaller：

```powershell
pip install pyinstaller
```

2) 在仓库根目录运行打包（单文件 exe）：

```powershell
pyinstaller --noconsole --onefile -n "小说人物扮演指令生成器" app\main.py
```

3) 生成的 exe 在 `dist/` 目录。

若你希望带一个图标、或把 `artifacts/` 作为默认输出目录并自动创建，都可以继续增强。

## 隐私与合规

**Key**：API Key 不保存到本地文件。

- 小说文本会被抽取片段并发送给模型进行归纳（这是“走 API 全自动”的必要条件）。如果你处理的是无授权文本，存在版权风险；建议只用于自写/已授权内容。
